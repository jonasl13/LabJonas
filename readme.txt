Kontrollspørsmål
Spm.0: Hva kan JavaScript brukes til?
JavaScript kan brukes til å tilføre dynamiske elementer til nettsider. Det mest normale er fjerne eller legge til tekst avhengig av hvor på nettsiden du klikker, 
eller åpne popuvinduer. 

Spm.1: Den største forskjellen mellom CSS Class og CSS id, er at CSS id er unik, mens CSS Class ikke er det. Dvs at du kan bruke CSS Class så mange ganger du vil, 
mens med CSS id så begrenser du deg til et spesifikt element. Det er veldig likt med forskjellen på serienummer og strekkode. 
Strekkoden identifiserer hva slags type vare det er, men det finnes flere som er helt like. 
Serienummeret derimot er helt unikt, det finnes bare en vare som har det serienummeret. CSS 

Spm.2: Designet til versjon 2 (rullegardina) er desidert best og ryddigst når det ikke er snakk om så alt for mange elementer å velge mellom. 
Det tar minst plass og er enklest og bruke når det er få elementer. Men når det blir 100 eller flere elementer å velge mellom, 
så vil jeg si at versjon 1(liste med farget tekst) er en bedre løsning. Dette er grunnet simplisiteten som versjon 1 har. Den tar større plass, 
men det er enklere å finne frem pga de forskjellige fargene, og størrelsen på teksten.

Spm.3: Wikipedia har satt opp den lange listen med programmeringsspråk på en lett og oversiktlig måte. Programmeringsspråkene ligger i listeform nedover siden, 
og er rangert alfabetisk. Dette gjør at det er lett å finne fram til det man er ute etter. I tillegg så står alfabetet horisontalt både på toppen av siden, 
og i bunn. Der kan du trykke på forbokstaven til ønsket programmeringsspråk for så og bare få fram alternativer med denne forbokstaven. 
Dette gjør at du fort kan finne frem til ønsket språk. Du har også valget mellom å rangere programmeringsspråkene kronologisk, kategorisk, 
eller generasjonsmessig. Dette gjør at hver enkelt bruker kan bruke den måten han/hun syntes er best til å navigere på siden. 
Når du har funnet frem til ønsket programmeringsspråk, så ligger det selvfølgelig en link til en annen wikipedia side der du kan lese mere om det valgte språket. 

Spm.4: <div> brukes til seksjoner på en nettside, mens <span> brukes midt i en tekst for å feks forandre farge.
